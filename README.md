# Complete-NLP-Session
## 🤖 Natural Language Processing (NLP) Fundamentals
This repository is a comprehensive collection of Jupyter Notebooks detailing the core concepts and implementation of Natural Language Processing using the NLTK (Natural Language Toolkit) library in Python. These notebooks serve as a practical guide for anyone looking to master text preprocessing and feature engineering.
## 🛠️ Tech Stack & Requirements
To run these notebooks locally, you will need:
Python 3.x
Jupyter Notebook / Google Colab
NLTK Library (pip install nltk)
Pandas (pip install pandas)
## 📧 Email & SMS Spam Deduction System
This repository contains a full-stack Machine Learning project focused on Natural Language Processing (NLP). The system is designed to classify text messages as Ham (normal) or Spam with a verified accuracy of 98.0%.
## 🚀 Project Overview
The project is divided into two main parts:
Research & Development: A series of Jupyter Notebooks documenting the experimentation with various NLP techniques.
Deployment: A professional Streamlit web application (webapplication.py) that allows users to check messages in real-time.
## 🛠️ Features
Advanced Preprocessing: Uses WordNetLemmatizer and stopwords filtering for high-quality data cleaning.
High Accuracy: The model is trained to reach a 98.0% accuracy rate.
Modern UI: A clean, blue-themed web interface with a sidebar for developer info and model stats.
Real-time Analysis: Immediate feedback on whether a message is "Safe" or "Spam."
